# Nerdy Gamer Tools (NGT)

> PHP-based gaming utility site with a whitelist router, Bootstrap 5, and Font Awesome.

## Overview

NGT is the main [Nerdy Gamer Tools](http://dev.nerdygamertools.com/) website. Pages are served via a `?page=` whitelist router; shared engine pieces live under `engine/` and individual page scripts under `pages/`.

## Requirements

- PHP
- A web server (Apache, Nginx, etc.)

## Setup

1. Deploy the project folder to your web host.
2. Point the document root at the project directory.
3. Open the site in a browser.
